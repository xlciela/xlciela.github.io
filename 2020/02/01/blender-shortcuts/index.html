BLENDER 2.8 常用快捷键总结

Modeling

界面/视图/图层操作:

  • T: 工具列表
  • N: 右侧明细
top back front back right left
7 ctrl + 7 1 ctrl + 1 3 ctrl + 3
  • 5: 切换正交/透视
  • [2,4,6,8]: 旋转视图
  • Shift + 7: 切换至以当前选中激活面的法线方向为视线方向的视图
  • M: 移动对象到指定collection
  • [1,2,3,...] ( object mode) : 查看指定scene collection内容
    • Shift + [1,2,3,...]: 同时查看指定collection内容
    • H: 隐藏选定对象
    • Shift + H: 仅显示选定对象
    • Alt + H: 取消隐藏
    • 鼠标停留在场景集合窗口内，按Alt + H: 显示所有scence collection内容
    • Ctrl + H: 控制需要显示的层级
    • [+/-]: 展开/收缩 场景集合文件目录
  • /: 局部/全局切换
    • Z: 着色方式
  • Shift + E + n: 边线折痕
    • Shift + E: Edge crease

物体基本操作:

  • W: [物体、点、线、面、…] 对应上下文菜单
  • Shift + A:创建对象
  • F9 : 微调参数
  • Shift + D: 复制对象
    • Alt + D: 关联复制 (物体模式下)
  • Ctrl + M: 按选定轴向镜像
  • Ctrl + J: 合并对象
  • P:分离对象
  • Ctrl + P: 设置父级
    • Shift + S: 对齐
    • Shift + R : Repeat previous step
    • G/R/S: 变换: [移动 / 旋转 / 缩放]
      • G G: 滑移——受限制的移动线或面
        • GG + C : 约束下滑移u
    • Alt + [G/R/...]: 复位变换
    • <: 修改变换坐标系
      • +: 可以设置[点, 线, 面]的法向基准的坐标系作为变换坐标系
  • >: 修改轴心点
  • Alt + N: 法向菜单
  • F2: 重命名
  • 选取操作:*
  • Ctrl + [+/-]: 扩展/缩小选区
  • Shift + S: 选择相似对象
  • Alt+Selecting: 循环选择
    • Ctrl + Alt+Selecting : 并排选择
  • B: 框选
  • C: 刷选
    • 按住中键可以减选
  • Select ion+L: 扩展选区至整个拓扑对象
  • Ctrl + I : 反选
  • 选择一个点，再 ctrl 选择另一个点：自动选择最短路径
  • 选择一个物体，double* A: 以选中物体为激活对象的集合
    点线面、变换:
点 线 面
选择 1 2 3
加/减选 Shift + 1 Shift + 2 Shift + 3
元素对应命令 Ctrl + V Ctrl + E Ctrl + F
  • X: delete
    • Ctrl + X: 融并选区
  • Ctrl + B: 倒角 ( edge )
    • Shift + Ctrl + B: Bevel ( vertex-only )
  • Ctrl + R: 环切
  • Alt + M : 焊接 ( merge )
  • I: 内插面
  • K: 裁剪?
  • 连接之F & J :
    • F: 元连接 ( 导轨情景下可以按住 F 使其按方向顺次连接
    • J : 二阶连接，在基础上进一步按选择方向连接
  • S + [x, y, z] + 0: 元素沿选择轴向对齐
    • 对齐基准的选择：按>选择对齐基准
  • Alt + E: 按选定方式挤出(法向挤出, etc)
  • Alt + S: 法向缩放
    • 按 S 开启均等缩放、按住 Alt 开启均等缩放
  • Shift + Alt + S: 球形化
  • O : 开启/关闭衰减编辑
  • V : 断离
    • Alt + V: 撕开选中的点/线 并填充 3d
    • Alt + D: 断离并延长 2d
  • Y: 从物体上拆分选中的元素
  • Shift + V : 沿网格滑移顶点
    • = G G
  • Shift + Tab: 开关吸附工具
    • 按 Ctrl : 暂时开关吸附工具
    • N 活动工具中开启 自动合并 则阀值内吸附的元素自动合并
  • Shift + N: 重新计算法线

Shading

  • Shift + A: 新建
  • Shift + D: 复制
  • Shift + Right_click: 添加节点

Camera setting & Rendering

  • F12: 查看渲染结果
  • Shift + S : save the rendering image
  • 0: (toogle) 在视图中查看相机看到的内容
  • Ctrl + Shift + 0: 于当前视图定位摄像机
  • Shift + `: Navigation, 第一视角操作
  • Contrl + Shift + left-click:preview the effect of the node
  • Contrl + G: Build a group
  • Tab: unfold the group

Sculpting

  • F: 调整画笔大小
  • Shift + F: 调整画笔强度
